Preparing an SSA run with fastgssa.

2019-06-21

library(fastgssa)
library(ggplot2)

The Decaying-Dimerization Reaction Set consists of three species and four reaction channels,

[ \begin{aligned} s_1 & \overset{c_1}{\rightarrow} 0 \ s_1 + s_2 & \overset{c_2}{\rightarrow} s_2 \ s_2 & \overset{c_3}{\rightarrow} s_1 + s_2 \ s_2 & \overset{c_4}{\rightarrow} s_3 \end{aligned} ]

Define parameters

params <- c(c1 = 1.0, c2 = 0.002, c3 = 0.5, c4 = 0.04)

Initial state vector

initial_state <- c(s1 = 10000, s2 = 0, s3 = 0)

State-change matrix

nu <- matrix(
  c(
    -1, -2, +2,  0,
    0, +1, -1, -1,
    0,  0,  0, +1
  ),
  nrow = 3,
  byrow = TRUE
)

Propensity functions

propensity_funs <- c(
  "r1 = c1 * s1",
  "r2 = c2 * s1 * s1",
  "r3 = c3 * s2",
  "r4 = c4 * s2"
)

Run simulation with direct method

out <- 
  ssa(
    initial_state = initial_state,
    propensity_funs = propensity_funs,
    nu = nu,
    params = params,
    method = ssa_direct(),
    final_time = 10,
    census_interval = .01,
    verbose = TRUE
  )
## Running SSA direct with console output every 1 seconds
## Start time: CURRTIME
## walltime: 0, simtime: 0
## SSA finished!

print(out$stats)
